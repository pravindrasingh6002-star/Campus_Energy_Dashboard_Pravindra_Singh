# Campus Energy Dashboard – Capstone Project

## Objective
This project analyzes campus-wide building electricity usage and creates a visual dashboard with daily/weekly consumption trends. It is the final Capstone Project for the course **Programming for Problem Solving using Python**.

---

## Features
✔ Automatic CSV ingestion from /data/  
✔ Exception handling & metadata extraction  
✔ Daily + weekly aggregations  
✔ Building-wise summary (mean, min, max, total)  
✔ Object-Oriented modeling (Building, MeterReading, Manager)  
✔ Dashboard (Line + Bar + Scatter) in a single figure  
✔ Export of cleaned dataset + summary CSV + summary.txt
